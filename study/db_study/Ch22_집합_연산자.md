# 공부한 내용


집합 연산자의 종류는 다음과 같다.

- UNION
- UNION ALL
- INTERSECT
- EXCEPT

!https://dataonair.or.kr/publishing/img/knowledge/SQL_205.jpg

아래와 같은 형태로 UNION 연산을 이용하면 JOIN을 사용하지 않고 2개의 테이블을 조회할 수 있다.

```sql
SELECT PLAYER_NAME 선수명, BACK_NO 백넘버 
FROM PLAYER 
WHERE TEAM_ID = 'K02' 

UNION 

SELECT PLAYER_NAME 선수명, BACK_NO 백넘버 
FROM PLAYER 
WHERE TEAM_ID = 'K07' ORDER BY 1;
```
