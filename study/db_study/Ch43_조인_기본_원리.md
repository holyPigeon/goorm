## 1. NL Join(Nested Loop Join)

### 기본 메커니즘

NL Join의 기본 메커니즘은 중첩 for문과 같다.

```java
for(i=0; i<100; i++){ 
	...
	for(j=0; j<100; j++){ 
		... 
	} 
}
```

예를 들어 조인할 테이블이 3개라고 한다면, 2개의 테이블을 먼저 조인하고 그 다음에 합쳐진 테이블과 나머지 한 테이블을 마저 조인하게 되는데, 이 과정에서 for문이 쓰이게 된다.
